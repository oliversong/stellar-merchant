Integrating with Astral

Include this script on your web page:

<script src="https://www.astralproject.us/static/js/astral.js" type="text/javascript"></script>'
<script>
  Astral = new Astral('{{g.user.authToken if g.user else "AUTH_TOKEN"}}','{{g.user.username if g.user else "USERNAME"}}');
</script>

Call this when someone wants to pay with a gift card:

Astral.payWithAstral('CARD_ID')

Call this when someone has bought a gift card:

Astral.buyAstralGiftCard('CARD_ID')
